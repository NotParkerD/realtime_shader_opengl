# realtime_shader_opengl

Realtime shader written in C++/OpenGL using GLSL for shader files

Controls
=====================================
(1) Gouraud Shading
(2) Phong Shading
(3) Silhouette Shading
(m) Next Material
(M) Previous Material
(l) Next Light
(L) Previous Light
(x/X) Move light in x-direction
(y/Y) Move light in y-direction
(z/Z) Move light in z-direction
